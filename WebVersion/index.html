Web-Version (highly experimental!)

This is an early and experimental build of the web-version for our game Circuit Cracker.
The current build is only tested on Firefox and Chrome for PC, many things such as UI and progress saving are not yet implemented and you can currently only play with mouse and keyboard. (so don't try it on mobile browsers...)

You also might experience lag and performance issues - just to be warned...
